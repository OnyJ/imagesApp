# Images App

## What is this ?

This application is a training to learn how to embed webpack in a Rails 6 application.
It will allow to stock images in the database.

## Get the application up and running

### The app

* Ruby `2.7.0`, Rails `6.0.3`
* Database : PostgreSql

### Deployment instructions

* run `bundle update`, `bundle install`
* run the server locally `rails server`
